# Orama Website (v1.0.2)

> **Tutoring, Simplified.**
> The official web platform for Orama, the student-led tutoring service at Beaumont High School.

![Orama Banner](public/ORamA.png)

## üìñ About The Orama Website

This repository contains the source code for the "Orama" website. The platform is designed to connect students with high-achieving peer tutors, provide access to academic resources, and offer a streamlined booking experience.

Built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**, it features a distinctive "Spatial Computing" aesthetic with high-fidelity glassmorphism, fluid animations (including a custom "Dynamic Island" inspired navigation and a "Side-Slide" menu), and a focus on premium, cinematic UI interactions.

### Key Features

- **Spatial UI**: Glass-morphic interfaces, live "active" gradients, and precise motion design.
- **Dynamic Navigation**: A dual-navigation system featuring a "Spatial Meta" header (Clock/Location) and a slide-out drawer menu.
- **Content Engine**: Markdown-based blog system for sharing academic tips and updates.
- **Performance**: Optimized for speed and responsiveness using Next.js App Router.
